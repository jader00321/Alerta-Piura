Widget reutilizable ListTile diseñado para mostrar acciones o enlaces de navegación dentro de las tarjetas de sección en PerfilScreen.

Muestra un icono a la izquierda, título, subtítulo opcional y un icono de chevron a la derecha. Es tappable a través de onTap. Puede tener un color personalizado para el icono y el texto.
