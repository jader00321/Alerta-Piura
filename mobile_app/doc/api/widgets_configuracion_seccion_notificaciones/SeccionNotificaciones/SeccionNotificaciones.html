Un widget reutilizable que muestra la sección de "Notificaciones" dentro de la pantalla de SettingsScreen.

Actualmente, este widget sirve como un punto de navegación. Muestra un ListTile que, al ser presionado, redirige al usuario a la pantalla de historial de notificaciones (/alertas o PantallaAlertas).

Nota: Este widget podría expandirse en el futuro para incluir SwitchListTiles que permitan activar/desactivar tipos específicos de notificaciones (ej. "Comentarios", "Estado de Reporte").
