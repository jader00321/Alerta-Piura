SocketService

SocketService class

Servicio Singleton para gestionar la conexión WebSocket (Socket.IO).

Maneja la conexión, autenticación, y la escucha de eventos en tiempo real desde el servidor. Proporciona métodos emit para enviar eventos y Streams públicos para que la UI reaccione a eventos específicos (como onStopSos).
