Widget que representa la barra de búsqueda superior en MapaView.

Se posiciona en la parte superior de la pantalla (generalmente dentro de un Stack). Incluye un campo de texto para buscar reportes (controlado por onSearchChanged) y un avatar de perfil CircleAvatar que funciona como botón.

El avatar muestra la inicial del usuario si está autenticado (AuthNotifier.isAuthenticated) o un icono de persona si es un invitado. Al tocarlo, navega a PerfilScreen o LoginScreen respectivamente.
