sendStatusToUI method

void sendStatusToUI()

Envía el estado actual del SOS (activo, ID, segundos) a la UI. Invocado por el listener 'getSosStatus'.

Implementation

void sendStatusToUI() {
  service.invoke('currentSosStatus', {
    'action': 'currentSosStatus',
    'isActive': _isActive,
    'alertId': _alertId,
    'seconds': _remainingSeconds,
  });
  debugPrint("BG_SERVICE: Estado actual enviado a UI -> isActive: $_isActive, alertId: $_alertId, remaining: $_remainingSeconds");
}
