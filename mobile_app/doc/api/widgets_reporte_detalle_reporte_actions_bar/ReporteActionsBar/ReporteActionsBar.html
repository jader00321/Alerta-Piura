Barra de acciones que se muestra debajo de la cabecera en la pantalla de detalle de reporte (ReporteDetalleScreen).

Muestra el contador de apoyos con un botón para dar/quitar apoyo, y el contador de comentarios. Si el usuario no está autenticado, el botón de apoyo redirige a la pantalla de login.
